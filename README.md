# PiP Anywhere

Force Picture-in-Picture on any video site — even ones that try to disable it like Netflix, Prime Video, Disney+, and more.

## How it works

One click (or `Alt+P`) injects a tiny script that sets `disablePictureInPicture = false` on all video elements and activates PiP on the largest visible video. Click again to exit PiP.

## Install

1. Clone or download this repo
2. Go to `chrome://extensions`, enable **Developer mode**
3. Click **Load unpacked** and select the extension folder
4. Done — click the icon or press `Alt+P` on any video page

## Usage

- **Click the extension icon** — starts PiP on the current tab
- **`Alt+P`** — keyboard shortcut to toggle PiP
- If PiP is already active, clicking again exits it
